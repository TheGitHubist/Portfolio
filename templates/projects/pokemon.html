Pokemon on Linux Console

Context

This project demonstrates networking concepts, database management, and multiplayer game development in Python. It's ideal for learning about client-server architectures, LAN communication, and building interactive console applications. Suitable for educational purposes, game development workshops, or as a foundation for more complex multiplayer systems.

Goal of the project

Pokemon on Linux Console is a multiplayer game project implemented in Python, featuring a client-server architecture. It recreates the classic Pokemon experience in a text-based console environment, allowing multiple players to connect and interact within a shared game world over a local area network (LAN).

Hard skills (what I did)

  • Developed a multiplayer client-server game in Python
  • Implemented user authentication and database management
  • Created real-time menu broadcasting across clients
  • Built modular architecture with separate client, server, and database

Technologies (what I used)

  • Python
  • Sockets for networking
  • SQLite for database
  • Threading for concurrent connections
