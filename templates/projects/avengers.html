Avengers.exe and Avengers.nsi

Project Overview

The Avengers project consists of an NSI installer file named AvengersInstaller, designed for deployment in a hypothetical apocalyptic scenario where an AI has taken control of the internet. The desktop application (Avengers.exe) was developed in C++ using Microsoft Visual Studio Code, while the installer was created with NSIS. The solution includes two RSA keys that provide an additional layer of security for sensitive files. The installer ensures seamless deployment of the app and keys, making it easy to integrate into broader security strategies.

How It Works

The desktop app allows users to encrypt any file by clicking the 'encode' button, utilizing the public RSA key for ciphering. Conversely, encrypted files can be decrypted using the private key. This RSA-based encryption ensures that only authorized users with the private key can access the original content, providing robust protection against unauthorized access in compromised environments.

Key Features

  • RSA Encryption: Secure file encryption and decryption using public and private keys
  • Desktop Application: User-friendly interface for easy file handling (developed in C++ with Microsoft Visual Studio Code)
  • NSI Installer: Automated installation of the app and RSA keys (created with NSIS)
  • Apocalyptic Scenario Ready: Designed for high-threat environments with AI control
  • Bonus Security Layer: Complements existing solutions for enhanced protection

Context and Use Cases

In a world where AI dominates the internet, securing sensitive data becomes paramount. This project addresses the need for offline, key-based encryption that operates independently of network vulnerabilities. It's ideal for activists, journalists, or individuals in dystopian scenarios requiring unbreakable file protection, serving as a critical tool for data sovereignty in extreme conditions.
