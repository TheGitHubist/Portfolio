Forum Hosted on the Dark Web

Context

This forum addresses the need for secure, anonymous online communities where privacy is paramount. It's suitable for whistleblowers, activists, journalists, and individuals seeking uncensored discussion spaces. The project demonstrates web development, security practices, and deployment on privacy-focused networks, serving as an educational tool for understanding Dark Web technologies and ethical considerations in online platforms.

Goal of the project

This project is a web-based forum designed to operate on the Dark Web, providing a platform for anonymous communication and community interaction. The Dark Web is a part of the internet that emphasizes privacy and anonymity, accessible via specialized software like Tor. It's important to note that while the Dark Web is often associated with illicit activities in popular media, it serves legitimate purposes such as protecting free speech, enabling secure communication for journalists and activists, and fostering communities that prioritize privacy over mainstream surveillance.

Hard skills (what I did)

  • Developed an asynchronous web forum with social features
  • Implemented user authentication and admin controls
  • Created channel-based discussions with posts, comments, and likes
  • Integrated friends system and private messaging
  • Deployed on Linux servers with Tor service

Technologies (what I used)

  • Python (Quart framework for asynchronous backend)
  • HTML/CSS/JavaScript for frontend
  • Tor for Dark Web hosting
  • Linux servers
  • Database for user management
