IDS Prototype for Linux

Context

Ideal for system administrators and security professionals managing Linux servers where file integrity is critical. Use cases include compliance monitoring, intrusion detection in production environments, and automated security auditing for sensitive directories.

Goal of the project

The IDS (Intrusion Detection System) Prototype is a Python-based solution designed for file integrity monitoring on Linux systems. It automates the process of tracking changes to critical files, ensuring system security and compliance by detecting unauthorized modifications.

Hard skills (what I did)

  • Developed a Python-based file integrity monitoring system
  • Implemented automated JSON file generation for monitored files
  • Created real-time change detection with precise timestamps
  • Integrated with Linux systemd service for continuous monitoring

Technologies (what I used)

  • Python
  • JSON for data storage
  • Linux systemd services
  • Bash scripting
