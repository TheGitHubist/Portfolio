Energy Consumption Prediction in Bordeaux

Context

This group project focuses on analyzing and predicting energy consumption patterns in Bordeaux using machine learning techniques. It addresses the growing need for sustainable energy management in urban areas, providing insights that can help optimize energy distribution and reduce waste. The project demonstrates practical applications of data science in environmental and urban planning contexts.

Goal of the project

The Energy Consumption Prediction project is a comprehensive machine learning initiative aimed at forecasting energy usage in Bordeaux. By collecting and analyzing historical data, the project develops predictive models that can anticipate consumption patterns, enabling better resource allocation and sustainability planning for the city.

Hard skills (what I did)

  • Gathered and processed energy consumption data from CSV files
  • Performed data cleaning and reliability checks using pandas and numpy
  • Developed predictive machine learning models with scikit-learn
  • Created data visualizations and graphs using matplotlib and seaborn
  • Built a Flask web application to display predictions and insights

Technologies (what I used)

  • Python
  • Pandas and NumPy for data processing
  • Scikit-learn for machine learning models
  • Matplotlib and Seaborn for data visualization
  • Flask for web application development
